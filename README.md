# Crafting-an-AI-Powered-HR-Assistant-A-Use-Case-for-Nestle-s-HR-Policy-Documents-Project
Developed an AI-powered conversational HR assistant that enables efficient retrieval of information from Nestlé HR policy documents. The system leverages NLP, embeddings, and a retrieval-based approach using GPT-3.5 to deliver accurate, context-aware responses.
